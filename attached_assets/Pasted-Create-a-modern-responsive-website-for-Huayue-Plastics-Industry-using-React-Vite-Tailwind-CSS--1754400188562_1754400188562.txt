Use local images (not external URLs) stored in public/images/.

Each section/page should have appropriate product or banner images to match the brand style of Huayue Plastics Industry.

Make sure images are responsive and optimized.

🤖 Telegram Bot Integration:
Add a TypeScript-based Telegram bot to the src/telegram/ directory.

Use the Telegram Bot API with the provided BOT_TOKEN and CHAT_ID.

When a user submits the Contact Us form:

Send the form details (name, email, phone number, message) to a specified Telegram channel or group via the bot.

Validate input before submission.

Show a success/failure message to the user after submission.
